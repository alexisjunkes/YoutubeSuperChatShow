Youtube SuperChat Show

The Youtube SuperChat Show is an opensource project aimed at bringing content creators closer to users. Displaying the superchats on the screen through a python application.

The app reads the users' names, sponsored value, and message, plus a few more configurable talks



How to install

First we must install the Python programming language

Download Python 3.13.0



Check the "Add Python.exe to PATH" box and continue the installation normally.
